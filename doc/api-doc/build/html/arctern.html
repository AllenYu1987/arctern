arctern.wrapper_func.ST_PrecisionReduce(geos, precision)[source]¶

Reduce the precision of geometry.

For every geometry in geometries, reduce the decimal places of its coordinates to the given number. The last decimal place will be rounded.

Note, the operation is performed NOT in "inplace" manner, i.e., new geometries in arrow::Array format will be construted and extra memory will be allocated.

Parameters

  • geos (pyarrow.array.string) -- Geometries organized as WKT.

  • geos -- The number to reduce the decimals places to.

Returns

Geometries organized as WKB.

Return type

pyarrow.array.string

Example
>>> import pandas
>>> import arctern
>>> data = pandas.Series(["POINT (1.333 2.666)", "POINT (2.655 4.447)"])
>>> rst = arctern.wkb2wkt(arctern.ST_PrecisionReduce(data, 3))
>>> print(rst)
    0    POINT (1.33 2.67)
    1    POINT (2.66 4.45)
    dtype: object

arctern.wrapper_func.ST_Crosses(left, right)[source]¶

Check whether geometries "spatially cross".

For every (left, right) pair with the same offset value in left and right, check if they "spatially cross". "Spatially cross" here means two the geometries have some, but not all interior points in common.
